                                    # （二）z-index 设置

                                    • 一般不直接设置具体数字，后续项目改版升级需要改动时比较麻烦
                                    • 可以对项目几个需要层级设置的区域进行划分，在相应区域内设置具体数字，如：
                                      • 弹出菜单 2 位数
                                      • 固定菜单 3 位数
                                    .topnav-detail {
  z-index: 50;
}

                                    # （三）outline 用法

                                    • 通常写 css 样式时，可以给 outline 设置样式
                                    • 达到 border 的效果且不占用像素值，完成当前元素的样式编写后再删除

                                    # （四）class 命名

                                    • 网页中属于广告的内容，最好不要出现与 广告 相关的单词
                                    • 部分浏览器会将 adv 等与广告相关的样式过滤掉

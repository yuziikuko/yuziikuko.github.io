                                                                                      六、MVVM模型

                                                                                      郁子小于 1 分钟约 108 字笔记Vue2尚硅谷张天禹
                                                                                      此页内容
                                                                                      • （一）概念
                                                                                      • （二）特点

                                                                                      # （一）概念

                                                                                      • M：模型 Model，对应 data 中的数据
                                                                                      • V：视图 View，模板代码
                                                                                      • VM：视图模型 ViewModel，Vue 实例对象

                                                                                      # （二）特点

                                                                                      • data 中所有的属性，最后都出现在 Vue 实例对象（vm）身上
                                                                                      • vm 身上所有的属性及 Vue 原型上所有属性，在 Vue 模板中都可以直接使用

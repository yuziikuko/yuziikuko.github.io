                                                                                      五、el与data的两种写法

                                                                                      郁子小于 1 分钟约 149 字笔记Vue2Vue.js尚硅谷
                                                                                      此页内容
                                                                                      • 1.el 的两种写法
                                                                                      • 2.data 的两种写法

                                                                                      # 1.el 的两种写法

                                                                                      • new Vue 时直接配置 el 属性el: '#root'。

                                                                                      • 先创建 Vue 实例，随后再通过vm.$mount('#root')指定 el 的值。

                                                                                      # 2.data 的两种写法

                                                                                      • 对象式
                                                                                      data: {
  name: "aaa";
}
                                                                                      • 函数式
                                                                                      data() {
	return {
		name: 'aaa'
	}
}
                                                                                      • 如何选择

                                                                                      使用组件时，必须使用函数式，否则会报错。

                                                                                      重要原则

                                                                                      由 Vue 管理的函数，一定不要写箭头函数，一旦写了箭头函数，this 就不再指向 Vue 实例，而是全局 Window 对象。
